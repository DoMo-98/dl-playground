# Web Planning Docs

This folder contains the initial planning documents for the interactive learning web inside `dl-playground`.

## Documents

- `product-definition.md` — product intent, learning goals, page rules, UX principles
- `content-map.md` — initial decomposition of the learning path into web learning units
- `mvp.md` — first usable slice of the product
- `architecture.md` — technical structure and implementation direction
- `backlog.md` — execution-ready task list for the first iterations
- `scaffold-plan.md` — proposed initial filesystem scaffold and first files to create

## Current Intent

Build a visual, interactive learning web for deep learning where each page combines:

- explanation
- visualization
- interaction
- immediate feedback

## Current Focus

The current focus is the web layer, not the experimental sandbox.
