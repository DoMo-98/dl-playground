# dl-playground web

Frontend for the interactive learning layer of `dl-playground`.

## Stack

- React
- Vite
- TypeScript
- React Router
- Tailwind CSS v4

## Current scope

Initial scaffold with:

- site shell
- learning path overview
- first interactive lesson: perceptron weighted sum and bias

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```
